# movieapp
MovieApp es una aplicacion Android de filtrado de pelicula  que se conecta a la API REST de The Movie DataBase (TMDB).

La misma cuenta con un DropDown menu que filtra las peliculas por categorias al modificar la URL a la que consulta.

La aplicacion podra navegar entre las paginas de una misma categoria al presionar en los botones "Siguiente" o "Anterior".

Cada una de esas películas podrá ser seleccionada y se desplegará una nueva vista con los detalles de la misma (género, idioma original, popularidad, fecha de estreno, entre otras).

Cuenta tambien con un Cache que guarda la informacion para evitar la consulta constante al servidor.
